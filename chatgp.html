Heart Stroke Risk Prediction System Using Machine Learning.

Predicting heart attacks is currently one of the most challenging challenges in the medical industry. Heart disease is one of the leading causes of mortality worldwide. Early detection and preventive measures can significantly reduce risks and improve patient outcomes. This project aims to develop a predictive model that determines the presence of heart disease based on various medical attributes. Using machine learning techniques, specifically Logistic Regression, the model analyzes key clinical parameters such as age, sex, chest pain type, heart rate, exercise-induced angina, ST depression, and thalassemia to assess the likelihood of heart disease. By leveraging Python and a user-friendly frontend, the system enables early diagnosis, timely intervention, and better healthcare efficiency. The testing results show that, when compared to other ML algorithms used, the Logistic Regression has the best accuracy, 85.54%.

Technologies Used

1. Frontend:

  • HTML5, CSS3, JavaScript
  • React.js

2. Backend:

  • Python
  • Flask (for API integration)

3. Machine Learning Model::

  • Logistic Regression (for heart disease prediction)
  • Pandas, NumPy (for data processing)
  • Scikit-Learn (for model implementation)
